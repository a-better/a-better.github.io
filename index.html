Game Register

게임을 등록하기 위해서는 2가지 방법이 존재한다. Mutli Player를 대상으로 하여 서버를 운영하는 게임과, 단순히 Single Player로 플레이하는 게임이 존재한다. 코드를 맡기는 경우, 현재는 Contact me를 통해 소스코드를 올려주면, 관련 프로그램을 업로드 하는 방식으로 실행된다. 차후에는 코드를 올리는 구역을 만들어 웹툰같이 웹 게임을 업로드 하는 방식으로 계획할 예정이다. 위에 그림은 rest api를 통해 게임을 등록하는 과정이다. Single Play 게임이라도 독자적으로 운영하고 싶은경우 위와 같이 설정하면 된다.

반면의 멀티플레이 게임 같은 경우는 서버를 직접 운영해야 하는 경우가 생겨, 링크만 제공해주고 Arachne Platform은 홍보나 카카오톡 같은 메신저를 연동하는 기능만 제공한다. 멀티 플레이어 게임의 경우에는 url만 넘겨주는 것이 아닌 IP와 Port 번호를 넘겨주게 되면 Arachne Platform은 이를 다이렉트 하는 기능을 제공한다.

위 그림은 Inner API 중 실제 API의 구조에 대해서 설명하고 있다. IP주소와 Port번호의 조합으로 서버의 IP 주소를 이해할 수 있는 구조로 되어있다.
