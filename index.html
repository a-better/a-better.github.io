Developer

Arachne is Opensource Project

누구든지, Arachne Project에 동참하실 수 있습니다. Arachne는 Github Page를 이용하여 소스를 관리하고 있으며 Web Game Platform에 관심있는 모두가 Arachne 프로젝트와 같이 성장할 수 있습니다. Contact Me를 통해 관리자와 언제든지 연락가능하며, Wiki Page를 통해 개발 및 개발자를 위한 활동을 장려하고 있습니다.
